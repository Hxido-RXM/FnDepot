Audiobookshelf是一个自架设的有声书和播客服务器。

特征
完全开源，包括安卓和iOS应用（测试版）
实时流媒体播放所有音频格式
搜索并添加播客，下载带有自动下载的剧集
多用户支持及自定义权限
按用户记录进度并同步设备
自动检测库更新，无需重新扫描
通过批量上传拖放文件夹上传书籍和播客
备份你的元数据+自动每日备份
渐进式网页应用（PWA）
网页版和安卓版的Chromecast支持
从多个来源获取元数据和封面艺术
章节编辑器和章节查找（使用 Audnexus API)
把你的音频文件合并成一个 m4b
将元数据和封面图片嵌入你的音频文件中
基础电子书支持和电子阅读器
电子出版、pdf、cbr、cbz
将电子书发送到设备（即Kindle）
开放RSS订阅源用于播客和有声书
